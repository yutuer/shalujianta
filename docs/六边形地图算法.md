# 六边形地图算法

## 坐标系统

项目使用 **axial 坐标** (Q, R) 系统。

## 六边形类型

- **扁顶六边形 (flat-top)**: 扁平面上下，顶点左右
- **顶点定义**（相对于 Control 左上角）:
  ```
  V0 = (60, 0)   - 顶部
  V1 = (110, 26) - 右上
  V2 = (110, 78) - 右下
  V3 = (60, 104) - 底部
  V4 = (10, 78)  - 左下
  V5 = (10, 26)  - 左上
  ```

## 边的定义

```
a (左上): V5→V0 = (10,26)→(60,0)
b (左中): V4→V5 = (10,78)→(10,26) - 垂直左边
c (左下): V3→V4 = (60,104)→(10,78)
d (右下): V2→V3 = (110,78)→(60,104)
e (右中): V1→V2 = (110,26)→(110,78) - 垂直右边
f (右上): V0→V1 = (60,0)→(110,26)
```

## 坐标转换公式

### Axial → 世界坐标
```
x = 100 * Q + 50 * R
y = 78 * R
```

### 世界坐标 → Axial
```
R = round(y / 78)
Q = round((x - 50 * R) / 100)
```

## HexCoord 方向向量

| 方向 | Q, R 增量 | 说明 |
|------|----------|------|
| North | (0, -1) | 正上方 |
| South | (0, 1) | 正下方 |
| NorthEast | (1, -1) | 右上 |
| NorthWest | (-1, 0) | 左上 |
| SouthEast | (1, 0) | 右下 |
| SouthWest | (-1, 1) | 左下 |

## 六边形尺寸

- 宽度：120px
- 高度：104px
- 中心点：(60, 52)
- 水平间距：100px
- 垂直间距：78px

## 示例：添加新六边形

假设要在 Hex7 (2, -2) 的右上方向添加新六边形：

```
Hex7 = (2, -2)
新坐标 = Hex7 + NorthEast = (2+1, -2-1) = (3, -3)

屏幕位置:
x = 100*3 + 50*(-3) = 150
y = 78*(-3) = -234

Control 位置 = (400+150, 300-234) = (550, 66)
```
